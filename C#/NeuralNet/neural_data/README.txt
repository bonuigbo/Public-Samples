The main program file is NeuralNet, and the helper library is MathNet.Numerics.

To run the program, add the extension .exe to NeuralNet, i.e. NeuralNet.exe,
and add the extension .dll to MathNet.Numerics, i.e. MathNet.Numerics.dll.

Do not modify the names or move any of the following folders
cluster_images,
test_data
test_images
training_data
training_images

They are required by the programto run. You can delete the images 
and data within the folders in order to generate your own samples.
